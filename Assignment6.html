1. HTML

Primary Use Cases Key Features/Benefits
Building the structure of web pages
  • Simple and easy to learn
  • Compatible with all browsers
  • Allows for semantic markup

2. CSS

Primary Use Cases Key Features/Benefits
Used for styling of web pages
  • Easy for the user to customize the online page
  • Web developers need to use few lines of programming for every page improving site speed.
  • It helps to form spontaneous and consistent changes.

3. JavaScript

Primary Use Cases Key Features/Benefits
To transform web pages from static to dynamic and bring interactivity into webpages.
  • JavaScript having the ability to support all modern browsers and produce an equivalent result.
  • JavaScript is employed everywhere on the web.
  • It gives the power to make rich interfaces.
