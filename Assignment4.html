HTML Tags

  1. !DOCTYPE : Defines the document type

  2. html : Defines an HTML document

  3. head - Contains metadata/information for the document

  4. title - Defines a title for the document

  5. h1 to h6 - Define HTML headings

  6. p - Defines a paragraph

  7. br - Inserts a single line break

  8. hr - Defines a thematic change in the content

  9. b - Defines bold text

  10. em : Defines emphasized text

  11. mark - Defines marked/highlighted text

  12. pre - Defines preformatted text

  13. progress - Represents the progress of a task

  14. q - Defines a short quotation

  15. s - Defines text that is no longer correct

  16. samp - Defines sample output from a computer program

  17. small - Defines smaller text

  18. strong - Defines important text

  19. form - Defines an HTML form for user input

  20. input - Defines an input control

  21. button - Defines a clickable button

  22. select - Defines a drop-down list

  23. option - Defines an option in a drop-down list

  24. output - Defines the result of a calculation

  25. datalist - Specifies a list of pre-defined options for input controls
